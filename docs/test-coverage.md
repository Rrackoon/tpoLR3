## Примечания

- Для динамических результатов WolframAlpha часть данных рендерится в `img` с `alt`, поэтому проверки идут не только по тексту DOM, но и по `alt` графических блоков.
- Все локаторы построены через `XPath`, без опоры на `id`.
- Код страниц и Selenium-инфраструктуры вынесен в `src/main/java/ru/itmo`, а тесты оставлены в `src/test/java/ru/itmo`.
